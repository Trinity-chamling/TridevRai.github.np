what it does

Online Bazar is used to display products according to category and make users able to buy it online.

Technology Used

Spring MVC, hibernate, Maven, JSP, JSTL, HTML/CSS/JS

Technical Description of the project

It is developed using Spring framework along with hibernate. The system contains 7 models namely; Category, Product, Cart, Order, Payment, UserDetail, and UserRole. All of these models are used to control the data in the web application. Category and Product modules can be used to perform CRUD operation on the database. Since this application is currently under development all the modules are not implemented.
