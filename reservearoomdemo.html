Reserve a Room

A hotel room booking web application.

what it does

used for showcasing services provided by a hotel and give platform to book available room.

Technology Used

Spring boot, hibernate,Maven, JSP, JSTL, HTML/CSS/JS

Technical Description of the project

The web application is build on MVC pattern. It contains five models namely, Room, Reservation, UserDetail, UserRole, CheckOut. All of these are used to hold the application data and transfer to the database. Room module can be used to perfrom CURD operation on the database. Since this app is currently being developed all the modules are not fully functional.
