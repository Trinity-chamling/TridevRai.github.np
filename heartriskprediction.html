Heart Risk Prediction

A machine learning web application to predict heart disease risk.

what it does

The machine learning application is used to predict risk of having heart disease by taking medical tests result as input.

Technology Used

PHP, SQL, Apache Server, HTML/CSS/JS

Technical Description of the project

The application uses Naive Bayes Algorithm to calculate prediction for new inputs. Naive Bayes is a supervised learning algorithm, it uses training dataset to train the classifier module. This training helps the module to predict new outcome for the provided input.
The prediction is binary prediction. This type of approach is clalled Classification in Data Mining world.
Through Admin module, admin can view feedbacks and users, add training dataset and doctor information. Users can enter value in the form field to get prediction.
